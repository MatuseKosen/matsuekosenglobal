img/ — Place your website images here
======================================

Recommended images to add:
  - hero-bg.jpg          (homepage hero background, ~1920×1080px)
  - about-director.jpg   (director/staff photo, ~400×400px)
  - campus-1.jpg         (campus photos, ~800×600px)
  - campus-2.jpg
  - news/                (subfolder for news article images)
  - events/              (subfolder for event photos)

Image tips for Google Drive hosting:
- Use .jpg for photos (smaller file size)
- Use .png for logos/icons with transparency
- Use .webp for best compression (modern browsers support it)
- Keep individual images under 1MB for fast loading
- Reference them in HTML as: <img src="../../img/hero-bg.jpg" alt="...">
  (adjust the ../ prefix based on how deep your HTML file is)
